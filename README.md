# Neo Domino Cards 🎴

Un catálogo web responsivo para una tienda de cartas coleccionables, desarrollado como trabajo práctico para la materia **Aplicaciones Web Cliente 2026**.

## 🚀 Tecnologías Utilizadas
Este proyecto está construido estrictamente con tecnologías frontend nativas (sin frameworks), aplicando las mejores prácticas de maquetado:
- **HTML5:** Estructura semántica.
- **CSS3:** Diseño moderno utilizando CSS Grid, Flexbox, un sistema de diseño propio mediante Variables (`:root`) y animaciones (`@keyframes`).

## 🌟 Características
- **Diseño Responsive (Mobile-First):** Adaptable perfectamente a celulares, tablets y monitores de escritorio mediante Media Queries.
- **UI/UX Moderno:** Efectos de transición suaves (`hover`), tarjetas con elevación (`transform`) y un layout de navegación superior fijo (`fixed`).
- **Catálogo y Detalle:** Flujo de navegación completo desde la vista general de productos hacia el detalle de cada carta.
- **Formularios:** Interfaz de reseñas integrada para los usuarios.

## 🛠️ Cómo ejecutar el proyecto
1. Clona este repositorio en tu máquina local.
2. Navega hacia la carpeta `app`.
3. Abre el archivo `index.html` en tu navegador web de preferencia.
   *Tip: Para una mejor experiencia de desarrollo, se recomienda utilizar la extensión **Live Server**.*
